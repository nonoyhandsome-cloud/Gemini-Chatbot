# 🚀 Setup and Run the Gemini Chatbot App
1. Install Node.js: Download the LTS version from nodejs.org. Confirm it’s ready by running node -v and npm -v in your terminal.

2. Extract & Open: Unzip your project files and open the folder in VS Code.

3. Install Dependencies: Open the VS Code terminal and run: npm install

4. Configure API Key: Open the .env file and replace your_api_key_here with your key from Google AI Studio.

5. Launch: Start the server by running: npm start

6. View: Go to http://localhost:3000 in your browser.

Note: If npm start throws an error, try node server.js directly.
